# hooks Directory Structure

## Overview
Custom React hooks for business logic, data management, and external integrations.

## Core Hooks

### Data Management
- **useAuth.ts** - Authentication and user state
- **useInventory.ts** - Inventory CRUD operations
- **useShoppingList.ts** - Shopping list management
- **useExpiryNotifications.ts** - Expiry tracking

### Integration Hooks
- **useBarcodeScanner.ts** - Barcode scanning
- **useSpoonSureProfile.ts** - Dietary preferences
- **useVigilSettings.ts** - Device configuration
- **useEbayNudge.ts** - Resale recommendations

### UI Hooks
- **use-toast.ts** - Toast notifications
- **use-mobile.tsx** - Mobile detection
