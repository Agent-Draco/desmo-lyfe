# src Directory Structure

## Overview
Main application source code directory containing all React components, hooks, and business logic.

## Subdirectories

### components/
React UI components and views
- **views/** - Page-level components
- **ui/** - Reusable UI components (shadcn/ui)

### hooks/
Custom React hooks for business logic and data management

### pages/
Top-level page components for routing

### lib/
Utility functions, helpers, and business logic

### contexts/
React context providers for global state

### integrations/
External service clients (Supabase, Vigil)

### assets/
Static assets (images, icons)

## Key Files
- **main.tsx** - Application entry point
- **App.tsx** - Main app component with routing
- **vite-env.d.ts** - Vite environment types
