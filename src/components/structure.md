# components Directory Structure

## Overview
React UI components organized by functionality and reusability.

## Subdirectories

### views/
Page-level components for different application views
- **HomeView.tsx** - Dashboard overview
- **InventoryView.tsx** - Inventory management
- **ShoppingListView.tsx** - Shopping list
- **ScanView.tsx** - Scanning interface
- **FamilyView.tsx** - Household management
- **NudgesView.tsx** - AI recommendations
- **CommView.tsx** - Community marketplace
- **SettingsView.tsx** - Application settings

### ui/
shadcn/ui component library
- Reusable UI components (buttons, forms, etc.)
- Consistent design system

## Root Components
Main application components
- **Header.tsx** - Application header
- **GlassNav.tsx** - Bottom navigation
- **GlassCard.tsx** - Glassmorphism card
- **RecipeDrawer.tsx** - Recipe display
- **NudgeFeed.tsx** - Notification feed
