# ui Directory Structure

## Overview
shadcn/ui component library - reusable UI components.

## Components
Complete set of UI components including:
- Forms (button, input, select, etc.)
- Layout (card, dialog, sheet, etc.)
- Navigation (tabs, breadcrumb, etc.)
- Feedback (toast, alert, etc.)
- Data display (table, badge, etc.)

## Purpose
- Consistent design system
- Reusable components
- Accessibility compliant
- Styled with Tailwind CSS
