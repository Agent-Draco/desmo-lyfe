# pages Directory Structure

## Overview
Top-level page components used in React Router configuration.

## Pages

### Core Pages
- **Index.tsx** - Main dashboard (authenticated users)
- **Auth.tsx** - Login and signup
- **Onboarding.tsx** - New user setup
- **Landing.tsx** - Marketing landing page

### Feature Pages
- **VigilSetup.tsx** - Device configuration
- **Feedback.tsx** - User feedback form
- **NotFound.tsx** - 404 error page

## Routing
Pages are configured in `App.tsx` with lazy loading for performance.
